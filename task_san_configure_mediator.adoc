[role="lead"]
使用 System Manager 配置调解器服务器以用于自动故障转移。如果您尚未将自签名 SSL 和 CA 替换为经过第三方验证的 SSL 证书和 CA ，也可以将其替换为。

.步骤
. 导航到 * 保护 > 概述 > 调解器 > 配置 * 。
. 单击 * 添加 * ，然后输入以下调解器服务器信息：
+
** IPv4 地址
** Username
** Password
** 证书

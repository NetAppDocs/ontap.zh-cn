= 指定用于 SVM DR 关系的聚合
:icons: font
:imagesdir: ../media/


[role="lead"]
创建灾难恢复 SVM 后，您可以将 `aggr-list` 选项与 `vserver modify` 命令结合使用，以限制用于托管 SVM DR 目标卷的聚合。

.步骤
. 修改灾难恢复 SVM 的 aggr-list 以限制用于托管灾难恢复 SVM 卷的聚合：


`ClusterDR ：： > vserver modify -vserver svmDR -aggr-list < 逗号分隔列表 >`

= 关于 SnapMirror 卷复制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
传统上， ONTAP 复制技术可满足灾难恢复（ DR ）和数据归档的需求。在 ONTAP 9.3 中，这些技术的组合方式允许您在同一目标卷上配置灾难恢复和归档。

= 管理Snapshot预留概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snapshot预留_会为快照预留一定百分比的磁盘空间、默认情况下为5%。由于快照会在快照预留空间用尽时使用活动文件系统中的空间、因此您可能需要根据需要增加快照预留。或者、您也可以在预留空间已满时自动删除快照。

= SnapMirror 关系的兼容 ONTAP 版本
:icons: font
:imagesdir: ../media/


[role="lead"]
在创建 SnapMirror 数据保护关系之前，您应验证源卷和目标卷是否正在运行兼容的 ONTAP 版本。

[NOTE]
====
SVM 复制不支持与版本无关的功能。

====

= Hyper-V 和基于 SMB 的 SQL Server 无中断运行的含义
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Hyper-V 和基于 SMB 的 SQL Server 无中断运行是指通过这些功能的组合，可以使应用程序服务器和包含的虚拟机或数据库保持联机状态，并在执行多项管理任务期间提供持续可用性。这包括存储基础架构的计划内和计划外停机。

支持通过 SMB 对应用程序服务器执行无中断操作的操作包括：

* 计划内接管和交还
* 计划外接管
* 升级
* 计划内聚合重新定位（ ARL ）
* LIF 迁移和故障转移
* 计划内卷移动

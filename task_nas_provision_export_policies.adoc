[role="lead"]
使用导出策略启用 Linux 客户端对 NFS 服务器的访问。

此操作步骤将为创建或修改导出策略 link:task_nas_enable_linux_nfs.html["已启用 NFS 的现有 Storage VM"]。

.步骤
. 在 System Manager 中，单击 * 存储 * > * 卷 * 。
. 单击启用了 NFS 的卷，然后单击 * 更多 * 。
. 单击 * 编辑导出策略 * ，然后单击 * 选择现有策略 * 或 * 添加新策略 * 。

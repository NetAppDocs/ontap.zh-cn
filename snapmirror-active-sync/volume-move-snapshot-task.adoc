= 卷移动参考快照显示为最新
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


.问题描述 ：
[role="lead"]
对一致性组卷执行卷移动操作后、卷移动引用快照可能会错误地显示为SnapMirror关系的最新快照。

您可以使用以下命令查看最新快照：

`snapmirror show -fields newest-snapshot status -expand`

.解决方案 ：
手动执行 `snapmirror resync` 或者、等待卷移动操作完成后下一次自动重新同步操作。

= 在目标卷上设置隔离时， ONTAP SnapMirror删除操作失败
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
如果任何目标卷设置了重定向防护，而SnapMirror删除操作失败，请使用以下信息。

.问题描述 ：
如果任何目标卷设置了重定向隔离，则 SnapMirror 删除操作将失败。

.解决方案
执行以下操作以重试重定向并从目标卷中删除隔离。

* SnapMirror 重新同步
* SnapMirror 更新

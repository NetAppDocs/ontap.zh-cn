= 网络端口要求
:allow-uri-read: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SnapMirror 活动同步复制有几个网络端口要求。

例如，复制中涉及的每个节点都需要监控其合作伙伴节点的运行状况，控制复制活动，并且必须复制数据本身。此外，ONTAP Mediator 或 ONTAP Cloud Mediator 需要与所有节点通信，以提供仲裁服务并协助故障转移。所有这些功能都有不同的端点和端口要求。

在大多数情况下，SnapMirror 活动同步复制将在没有端口或基于目标的限制的内部网络上运行。如果 IP 流量受到防火墙、路由器、网络拓扑或其他机制的限制，则应验证以下连接要求。

= 在删除 Snapshot 副本之前计算可回收空间
:icons: font
:imagesdir: ../media/


[role="lead"]
从 ONTAP 9.10.1 开始，您可以使用 System Manager 选择要删除的 Snapshot 副本，并在删除这些副本之前计算可回收空间。

.步骤
. 单击 * 存储 > 卷 * 。
. 选择要从中删除 Snapshot 副本的卷。
. 单击 * Snapshot 副本 * 。
. 选择一个或多个 Snapshot 副本。
. 单击 * 计算可回收空间 * 。

= 禁用本地帐户的TOTP机密密钥
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果本地用户丢失了基于时间的一次性密码(TOTP)密钥、则存储管理员必须先禁用丢失的密钥、然后用户才能创建新的TOTP密钥。

.关于此任务
只能使用集群管理员帐户执行此任务。

.步骤
. 禁用TOTP密钥：
+
[source, cli]
----
security login totp delete -vserver "<svm_name>" -username "<account_username>"
----

= 了解如何启用本地ONTAP帐户访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本地帐户是指帐户信息，公有密钥或安全证书驻留在存储系统上的帐户。您可以使用 `security login create`命令使本地帐户能够访问管理员或数据SVM。

= 配置 LDAP 或 NIS 服务器访问概述
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须先配置对 SVM 的 LDAP 或 NIS 服务器访问，然后 LDAP 或 NIS 帐户才能访问 SVM 。通过交换机功能，您可以使用 LDAP 或 NIS 作为备用名称服务源。

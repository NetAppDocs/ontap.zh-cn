= 为 MetroCluster 配置启用自动切换
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本主题提供了有关在还原 MetroCluster 配置后必须执行的其他任务的信息。

. 启用自动计划外切换： `metrocluster modify -auto-switchover-failure-domain auso-on-cluster-disaster`
. 验证MetroCluster配置： `metrocluster check run`

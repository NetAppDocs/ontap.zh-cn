= 还原 ONTAP 概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要将集群过渡到早期的 ONTAP 版本，必须执行还原。

本节中的信息将指导您完成还原前后应执行的步骤，包括您应阅读的资源以及应执行的必要还原前和还原后检查。


NOTE: 如果您需要将集群从 ONTAP 9.1 过渡到 ONTAP 9.0 ，则需要使用所记录的降级操作步骤 link:https://library.netapp.com/ecm/ecm_download_file/ECMLP2876873["此处"]。

= 在从 ONTAP 9.19.1 恢复之前，请删除第二个 S3 访问密钥
:allow-uri-read: 


[role="lead"]
如果您升级到 ONTAP 9.19.1 并向任何用户添加了第二个 S3 访问密钥，则必须先删除这些密钥，然后才能还原到较早的 ONTAP 版本。

.步骤
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["显示 S3 访问密钥"] 以确定是否已为任何 S3 用户添加了第二访问密钥。
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["删除第二个访问密钥"] 适用于所有 S3 用户。

= 在还原双节点和四节点 MetroCluster 配置之前禁用自动计划外切换
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
还原双节点或四节点 MetroCluster 配置之前，必须禁用自动计划外切换（ AUSO ）。

. 在 MetroCluster 中的两个集群上，禁用自动计划外切换： `MetroCluster modify -auto-switchover-failure-domain auso-disabled`

= 启用自动更新
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 System Manager 启用自动更新，并允许 ONTAP 在无需您干预的情况下下载并安装固件、系统和安全更新。

您可以使用 ONTAP CLI 仅管理 SP/BMC 固件更新。

* 详细了解link:../system-admin/methods-manage-sp-bmc-firmware-updates-concept.html["管理 ONTAP SP/BMC 固件更新"]。
* 如需详细了解 System Manager 和 CLI 管理自动更新之间的区别，请参阅知识库文章：link:https://kb.netapp.com/on-prem/ontap/Ontap_OS/OS-KBs/Why_do_GUI_and_CLI_show_different_status_for_ONTAP_automatic_updates_configuration["为什么 GUI 和 CLI 对 ONTAP 自动更新配置显示不同的状态？"]。


每个系统组件的自动更新的可用性取决于您的ONTAP版本。

[cols="25,75"]
|===
| ONTAP 版本 | 可用的自动更新 


| 9.16.1及更高版本  a| 
* 利用人工智能(ARP/AI)实现自主防兰功能
* ONTAP时区数据库
* 存储设备、磁盘和磁盘架的存储固件
* 用于服务处理器和BMC模块的SP/BMC固件
* 磁盘认证包(DQP)




| 9.13.1及更高版本  a| 
* ONTAP时区数据库
* 存储设备、磁盘和磁盘架的存储固件
* 用于服务处理器和BMC模块的SP/BMC固件
* 磁盘认证包(DQP)




| 9.10.1及更高版本  a| 
* 存储设备、磁盘和磁盘架的存储固件
* 用于服务处理器和BMC模块的SP/BMC固件
* 磁盘认证包(DQP)


|===
如果您的ONTAP版本没有自动更新、或者您未启用自动更新、则可以手动执行固件、时区数据库和安全更新。

.关于此任务
启用自动更新需要您接受 EULA 的条款并根据您的环境确认或配置每个系统组件的默认更新行为。

对于在 ONTAP 9.16.1 或更早版本中创建然后升级到 9.16.1 或更高版本的集群：

* 如果在升级前启用了自动更新功能，则升级后不会更改系统组件更新默认值。
* 如果在升级前未启用自动更新，则将应用 ONTAP 9.16.1 及更高版本中的系统组件更新默认设置。


自动系统组件更新默认设置为自动更新或显示通知，具体取决于您的ONTAP版本。在完成该过程之前，请确认这些设置适合您的环境。

这 https://www.youtube.com/watch?v=GoABILT85hQ["视频"^]简要显示了如何使用自动更新过程。

.开始之前
* 您必须拥有当前的支持授权。您可以在*系统详细信息*页面中的中查看此link:https://mysupport.netapp.com/site/["NetApp 支持站点"^]信息。
* 要启用自动更新、必须先启用AutoSupport with HTTPS。如果集群上未启用AutoSupport、或者使用另一个传输协议在集群上启用了、则可以在此过程中使用HTTPS启用AutoSupport。



NOTE: 默认情况下、AutoSupport OnDemand处于启用状态、如果配置为使用HTTPS传输协议向技术支持发送消息、则该功能正常运行。

* 从ONTAP 9.10.1开始、如果启用了自动更新、请确保与以下附加URL建立HTTPS连接：
+
** \https://support-sg-naeast.netapp.com
** \https://support-sg-nawest.netapp.com




.步骤
[role="tabbed-block"]
====
.System Manager - 9.16.1 9.161及更高版本
--
. 在System Manager中、选择*集群>设置*。
. 在 *软件更新* 下，选择 *启用*。
. 在*自动更新设置*下，为每种更新类型选择要采取的操作。
+
默认情况下，所有选项都选择 *自动更新*。您可以选择将默认选择修改为 *显示通知* 或 *自动关闭更新*。

. 选择 *I accept for current and all future update*，然后选择 *Save*。


--
.System Manager - 9.15.1 9.151及更早版本
--
. 在System Manager中、选择*集群>设置*。
. 在 *自动更新* 下，选择 image:icon_kabob.gif["菜单选择"]，然后选择 *启用*。
+
默认情况下，所有选项都选择 *自动更新*。您可以选择将默认选择修改为 *显示通知* 或 *自动关闭更新*。

. 选择 *I accept for current and all future updates* ，然后选择 *Save* 。


--
.命令行界面
--
您可以使用 ONTAP CLI 仅管理 SP/BMC 固件更新

. 启用自动固件更新：
+
[source, cli]
----
system service-processor image modify -node <node_name> -autoupdate true
----

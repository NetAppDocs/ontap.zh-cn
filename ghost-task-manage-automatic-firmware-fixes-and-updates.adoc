= 调整命名空间大小
:icons: font
:imagesdir: ../media/


从 ONTAP 9.10.1 开始，您可以使用 ONTAP 命令行界面增加或减小 NVMe 命名空间的大小。您可以使用 System Manager 增加 NVMe 命名空间的大小。您可以阅读有关此主题的更多信息 xref:./nvme/resize-namespace-task.adoc[此处]。

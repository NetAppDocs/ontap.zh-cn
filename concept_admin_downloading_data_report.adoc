= 下载网络数据以进行报告
:toc: macro
:hardbreaks:
:toclevels: 1
:toc: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/
:toc-position: content


[role="lead"]
从 ONTAP 9.8 开始，您可以下载 System Manager 中显示的有关网络的数据。

当您在 _List View_ 中显示信息时，可以单击 * 下载 * ，此时将下载显示的对象列表。

* 此列表将以逗号分隔值（ CSV ）格式下载。
* 仅下载可见列中的数据。
* CSV 文件名采用对象名称和时间戳的格式。

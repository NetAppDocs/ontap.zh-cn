= 概述
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
为与特定服务进行 ONTAP 通信预留了许多已知端口。如果存储网络环境中的端口值与 ONTAP 端口上的端口值相同，则会发生端口冲突。

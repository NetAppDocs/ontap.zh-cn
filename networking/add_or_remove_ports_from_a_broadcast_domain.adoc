= 在广播域中添加或删除端口
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在集群创建或加入操作期间，系统会自动创建广播域。您无需手动从广播域中删除端口。

如果通过物理网络连接或交换机配置更改了网络端口可访问性，并且某个网络端口属于其他广播域，请参见以下主题：

link:repair_port_reachability.html["修复端口可访问性"]

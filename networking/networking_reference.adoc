= 网络管理概述
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
此内容介绍了基本的存储网络管理。它介绍了如何配置物理和虚拟网络端口（ VLAN 和接口组），如何使用 IPv4 和 IPv6 创建 LIF ，如何管理集群中的路由和主机解析服务，如何使用负载平衡优化网络流量以及如何使用 SNMP 监控集群。

除非另有说明，否则此内容将适用场景所有版本的 ONTAP 9 。

在以下情况下，应使用此内容：

* 您希望了解 ONTAP 网络管理功能的范围。
* 您希望使用命令行界面，而不是 ONTAP 系统管理器。

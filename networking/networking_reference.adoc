= 网络管理概述
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以使用以下信息执行基本存储网络管理。您可以配置物理和虚拟网络端口（ VLAN 和接口组），使用 IPv4 和 IPv6 创建 LIF ，管理集群中的路由和主机解析服务，使用负载平衡优化网络流量以及使用 SNMP 监控集群。

除非另有说明，否则这些过程适用于所有版本的 ONTAP 9 。

在以下情况下，应使用这些过程：

* 您希望了解 ONTAP 网络管理功能的范围。
* 您希望使用命令行界面，而不是 System Manager 。

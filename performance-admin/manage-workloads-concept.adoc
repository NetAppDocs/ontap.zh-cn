= 管理工作负载
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 提供了一组功能强大的工具来管理工作负载。除了可以通过多种方式将工作负载转移到其他资源（添加控制器，移动卷或 LUN ，纵向扩展到闪存介质等）之外，您还可以使用存储服务质量（ QoS ）来保证关键工作负载的性能不会受到竞争工作负载的影响。

= 执行每日ONTAP Active IQ Unified Manager监控
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以执行每日监控，以确保没有任何需要关注的即时性能问题。

.步骤
. 从 Active IQ Unified Manager UI 中，转到 * 事件清单 * 页面以查看所有当前事件和已废弃事件。
. 从*view*选项中，选择 `Active Performance Events` 并确定需要执行的操作。

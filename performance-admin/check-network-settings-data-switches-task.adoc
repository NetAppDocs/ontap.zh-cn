= 检查数据交换机上的网络设置
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须在客户端，存储系统和交换机上保持相同的网络设置，以确保性能不会受到影响。网络中的所有组件都必须具有相同的 MTU 设置，才能获得最佳性能。

.步骤
. 对于数据交换机，请检查 MTU 大小是否设置为 9000 。
+
有关详细信息，请参见交换机供应商文档。

= 检查数据交换机上的网络设置
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须在客户端，服务器和存储系统上保持相同的网络设置，而 NIC 和交换机等网络设备一般应使用各自的最大值来确保性能不会受到影响。

网络中的所有组件都必须能够转发巨型 IP 帧（包括以太网在内的 9000 字节 IP ， 9022 字节），以获得最佳性能。数据交换机应至少设置为 9022 字节，但对于大多数交换机，典型值可能为 9216 。

对于数据交换机，请检查 MTU 大小是否设置为 9022 或更高。

+ 有关详细信息，请参见交换机供应商文档。

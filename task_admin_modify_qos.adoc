[role="lead"]
从 ONTAP 9.8 开始，在配置存储时，默认情况下会启用 QoS 。您可以在配置过程中禁用 QoS 或选择自定义 QoS 策略。  您也可以在配置存储后修改 QoS 。

.步骤
. 在 System Manager 中，单击 * 存储 * 并选择 * 卷 * 。
. 在要修改 QoS 的卷旁边，单击 image:icon_kabob.gif["菜单图标"] 并选择 * 编辑 * 。

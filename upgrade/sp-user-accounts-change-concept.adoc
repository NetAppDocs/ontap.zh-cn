= 可访问服务处理器的用户帐户的更改
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在ONTAP 9.8及更早版本中创建了可使用非管理员角色访问服务处理器(SP)的用户帐户、并升级到ONTAP 9.9.1或更高版本、则`-role`参数中的任何非管理员值都会修改为`admin`。

有关详细信息，请参见 link:../system-admin/accounts-access-sp-concept.html["可访问 SP 的帐户"]。

= 升级后验证网络配置
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.8 及更高版本会自动监控第 2 层可访问性。从 ONTAP 9.7x 或更早版本升级到 ONTAP 9.8 或更高版本后，您应验证每个 .network 端口是否可访问其预期广播域。

. 验证每个端口是否可访问其预期域：``network port reachability show -detail``
+
如果可访问性状态为 ok ，则表示端口对其分配的域具有第 2 层可访问性。

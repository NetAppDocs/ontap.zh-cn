= SnapLock 的升级注意事项
:icons: font
:imagesdir: ../media/


[role="lead"]
如果某些内核版本已被限定为错误的 SnapLock 版本或在这些版本中禁用了 SnapLock ，则 SnapLock 不允许下载这些版本。只有当节点具有 SnapLock 数据时，这些下载限制才适用。

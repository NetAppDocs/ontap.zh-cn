= 配置对SVM的SMB访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果尚未为 SMB 客户端访问配置 SVM ，则必须创建并配置新的 SVM 或配置现有 SVM 。配置 SMB 包括打开 SVM 根卷访问，创建 SMB 服务器，创建 LIF ，启用主机名解析，配置名称服务，如果需要， 启用 Kerberos 安全性。

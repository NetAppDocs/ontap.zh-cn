= 网络架构概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 数据中心实施的网络架构通常由集群互连，用于集群管理的管理网络和数据网络组成。NIC （网络接口卡）为以太网连接提供物理端口。HBA （主机总线适配器）为 FC 连接提供物理端口。

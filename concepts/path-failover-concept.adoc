= 路径故障转移概述
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 在 NAS 和 SAN 拓扑中管理路径故障转移的方式存在重要差异。链路出现故障后， NAS LIF 会自动迁移到其他网络端口。SAN LIF 不会迁移（除非您在发生故障后手动移动）。相反，主机上的多路径技术会将流量转移到同一 SVM 上的其他 LIF ，但会访问不同的网络端口。

= 评估物理存储要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在为客户端配置 NFS 存储之前，您必须确保现有聚合中有足够的空间来容纳新卷。如果没有，您可以向现有聚合添加磁盘或创建所需类型的新聚合。

.步骤
. 显示现有聚合中的可用空间：
+
`storage aggregate show`

+
如果聚合具有足够的空间，请在工作表中记录其名称。

+
[listing]
----
cluster::> storage aggregate show

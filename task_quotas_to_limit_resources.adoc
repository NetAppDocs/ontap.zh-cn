[role="lead"]
添加配额以限制配额目标可以使用的磁盘空间量。

您可以为配额设置硬限制和软限制。

硬配额会对系统资源施加硬限制；任何可能导致超过此限制的操作都将失败。软配额会在资源使用量达到特定级别时发送警告消息，但它们不会影响数据访问操作，因此您可以在超过配额之前采取适当的操作。

.步骤
. 单击 * 存储 > 配额 * 。
. 单击 * 添加 * 。

[role="lead"]
如果现有默认保护策略不适合您的需求，您可以使用 System Manager 创建自定义数据保护策略。

在源集群和目标集群上创建自定义保护策略。

.步骤
. 单击 * 保护 > 本地策略设置 * 。
. 在 * 保护策略 * 下，单击 image:icon_arrow.gif["箭头"]。
. 在 * 保护策略 * 窗格中，单击 image:icon_add.gif["此外"]。
. 填写必填字段。
. 单击 * 保存 * 。
. 在另一个集群上重复上述步骤。

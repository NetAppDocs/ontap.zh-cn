= 重新激活源 Storage VM

[role="lead"]
从 ONTAP 9.8 开始，您可以使用 System Manager 在发生灾难后重新激活源 Storage VM 。重新激活源 Storage VM 将停止目标 Storage VM ，并重新启用从源到目标的复制。

.步骤
. 选择所需的保护关系：单击 * 保护 > 关系 * 。
. 单击 image:icon_kabob.gif["kabob 图标"] 然后单击 * 重新激活源 Storage VM* 。
. 在 * 关系 * 下，通过查看保护关系的 * 传输状态 * 来监控源重新激活进度。

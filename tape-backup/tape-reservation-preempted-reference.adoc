= 已抢占磁带预留
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Tape reservation preempted`

* * 发生原因 *
+
磁带驱动器正在被其他操作使用，或者磁带已过早关闭。

* * 更正操作 *
+
确保磁带驱动器未被其他操作使用，并且 DMA 应用程序未中止作业，然后重试。

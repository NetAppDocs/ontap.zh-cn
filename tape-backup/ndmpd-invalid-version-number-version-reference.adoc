= ndmpd 版本号无效： version_number ``
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`ndmpd 版本号无效： version_number`

* * 发生原因 *
+
存储系统不支持指定的 NDMP 版本。

* * 更正操作 *
+
指定 NDMP 版本 4 。

= 正在迁移底层聚合
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Underlying aggregate under migration`

* * 发生原因 *
+
如果对正在迁移（存储故障转移或聚合重新定位）的聚合启动 SMTape 操作，则 SMTape 操作将失败。

* * 更正操作 *
+
在聚合迁移完成后执行 SMTape 操作。

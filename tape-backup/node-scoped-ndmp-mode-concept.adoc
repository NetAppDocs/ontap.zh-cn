= 什么是节点范围的 NDMP 模式
:icons: font
:imagesdir: ../media/


[role="lead"]
在节点范围的 NDMP 模式下，您可以在节点级别执行磁带备份和还原操作。从 8.2 升级到 8.3 后， Data ONTAP 8.2 中使用的 NDMP 操作模式将继续保留。

在节点范围的 NDMP 模式下，您可以在拥有卷的节点上执行磁带备份和还原操作。要执行这些操作，您必须在拥有卷或磁带设备的节点上托管的 LIF 上建立 NDMP 控制连接。

[NOTE]
====
此模式已弃用，将在未来的主要版本中删除。

= 磁带读取错误
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Tape read error`

* * 发生原因 *
+
磁带驱动器已损坏或磁带不包含完整备份。

* * 更正操作 *
+
如果磁带驱动器已损坏，请使用另一个磁带驱动器。如果磁带不包含完整备份，则无法还原数据。

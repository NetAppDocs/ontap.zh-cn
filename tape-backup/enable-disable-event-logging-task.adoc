= 启用或禁用事件日志记录
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以打开或关闭事件日志记录。

.步骤
. 要启用或禁用事件日志记录，请在 clustershell 中输入以下命令：
+
`*options _-option_name_ backup.log.enable _-option-value_ {on | off}*`

+
`on` 打开事件日志记录。

+
`off` 关闭事件日志记录。

+
[NOTE]
====
默认情况下，事件日志记录处于打开状态。

====

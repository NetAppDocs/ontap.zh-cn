= 磁带记录大小应为 block_Size1 ，而不是 block_size2
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Tape record size should be block_size1 and not block_size2`

* * 发生原因 *
+
为本地还原指定的阻止因素不正确。

* * 更正操作 *
+
使用重试还原作业 `block_size1` 作为分块系数。

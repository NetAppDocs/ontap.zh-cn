= 允许正在进行的最大转储或还原数（最大会话限制）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Maximum number of allowed dumps or restores _(maximum session limit)_ in progress`

* * 发生原因 *
+
正在运行的备份或还原作业的最大数量。

* * 更正操作 *
+
在当前正在运行的某些作业完成后重试此操作。

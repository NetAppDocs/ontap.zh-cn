= 磁带写入时出现介质错误
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`m磁带写入介质错误`

* * 发生原因 *
+
用于备份的磁带已损坏。

* * 更正操作 *
+
更换磁带并重试备份作业。

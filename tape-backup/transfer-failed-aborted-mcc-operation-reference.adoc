= 传输失败（由于 MetroCluster 操作而中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`传输失败（由于 MetroCluster 操作而中止）`

* * 发生原因 *
+
SMTape 操作因切换或切回操作而中止。

* * 更正操作 *
+
在切换或切回操作完成后执行 SMTape 操作。

= 了解合格的ONTAP磁带驱动器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须使用经过测试且发现可在存储系统上正常工作的合格磁带驱动器。您可以遵循磁带别名设置，也可以启用磁带预留，以确保在任何特定时间只有一个存储系统访问磁带驱动器。

合格的磁带驱动器是一种经过测试且可在存储系统上正常工作的磁带驱动器。您可以使用磁带配置文件来确定现有 ONTAP 版本中的磁带驱动器的资格。

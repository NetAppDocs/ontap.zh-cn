= 无法删除快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Failed to delete snapshot`

* * 发生原因 *
+
无法删除此自动快照、因为它正在由其他操作使用。

* * 更正操作 *
+
使用 `snap`命令确定快照的状态。如果不需要快照、请手动将其删除。

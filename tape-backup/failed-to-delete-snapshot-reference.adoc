= 无法删除快照
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`无法删除 Snapshot`

* * 发生原因 *
+
无法删除自动 Snapshot 副本，因为它正在由其他操作使用。

* * 更正操作 *
+
使用 `snap` 命令确定 Snapshot 副本的状态。如果不需要 Snapshot 副本，请手动将其删除。

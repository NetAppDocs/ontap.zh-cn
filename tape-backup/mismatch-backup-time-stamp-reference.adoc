= 备份时间戳不匹配
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Mismatch in backup time stamp`

* * 发生原因 *
+
磁带更改期间加载的磁带不属于备份集。

* * 更正操作 *
+
使用 `smtape restore -h` 用于验证磁带标头信息的命令。

= 备份时间戳不匹配
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`m备份时间戳不匹配`

* * 发生原因 *
+
磁带更改期间加载的磁带不属于备份集。

* * 更正操作 *
+
使用 `smtape restore -h` 命令验证磁带的标头信息。

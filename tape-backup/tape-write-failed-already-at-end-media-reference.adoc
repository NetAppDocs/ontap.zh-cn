= 磁带写入失败—新磁带已位于介质末尾
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`磁带写入失败—新磁带已位于介质末尾`

* * 发生原因 *
+
磁带上没有足够的空间来完成备份。

* * 更正操作 *
+
更换磁带并重试备份。

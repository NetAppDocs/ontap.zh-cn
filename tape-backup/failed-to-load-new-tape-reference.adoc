= 无法加载新磁带
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Failed to load new tape`

* * 发生原因 *
+
磁带驱动器或介质出错。

* * 更正操作 *
+
更换磁带并重试此操作。

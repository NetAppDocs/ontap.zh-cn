= 还原文件 < 文件名 > 失败
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Restore of the file file name failed`

* * 发生原因 *
+
如果对文件名与目标卷上 LUN 的文件名相同的文件执行 DAR （直接访问恢复），则 DAR 将失败。

* * 更正操作 *
+
重试此文件的 DAR 。

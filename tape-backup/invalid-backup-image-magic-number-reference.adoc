= 备份映像幻数无效
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Invalid backup image magic number`

* * 发生原因 *
+
备份映像不是 SMTape 映像。

* * 更正操作 *
+
如果备份映像不是 SMTape 映像，请使用具有 SMTape 备份的磁带重试此操作。

= 无法锁定转储所需的快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`无法锁定转储所需的快照`

* * 发生原因 *
+
为备份指定的 Snapshot 副本不可用。

* * 更正操作 *
+
使用其他 Snapshot 副本重试备份。

+
使用 `snap list` 命令查看可用 Snapshot 副本的列表。

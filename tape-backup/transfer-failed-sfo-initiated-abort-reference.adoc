= 传输失败（ SFO 启动的中止）
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`传输失败（ SFO 启动中止）`

* * 发生原因 *
+
SMTape 操作因存储故障转移（接管和交还）操作而中止。

* * 更正操作 *
+
在存储故障转移（接管和交还）操作完成后执行 SMTape 操作。

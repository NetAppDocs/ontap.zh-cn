= 磁带无序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`Tapes out of order`

* * 发生原因 *
+
用于还原操作的磁带序列中的第一个磁带没有映像标头。

* * 更正操作 *
+
加载包含映像标题的磁带，然后重试此作业。

= 传输失败（ CFO 启动的中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Transfer failed (CFO initiated abort)`

* * 发生原因 *
+
SMTape 操作因 CFO 聚合的存储故障转移（接管和交还）操作而中止。

* * 更正操作 *
+
在 CFO 聚合的存储故障转移完成后执行 SMTape 操作。

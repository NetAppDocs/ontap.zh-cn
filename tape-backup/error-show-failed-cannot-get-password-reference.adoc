= 错误：显示失败：无法获取用户 "&lt ； username&gt" 的密码
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`错误：显示失败：无法获取用户 "< 用户名 >" 的密码`

* * 发生原因 *
+
NDMP 的用户帐户配置不完整

* * 更正操作 *
+
确保用户帐户与 SSH 访问方法关联，并且身份验证方法为用户密码。

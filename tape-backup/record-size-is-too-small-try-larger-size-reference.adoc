= 磁带记录大小太小。请尝试更大的大小。
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`磁带记录大小太小。请尝试更大的大小。`

* * 发生原因 *
+
为还原操作指定的分块系数小于备份期间使用的分块系数。

* * 更正操作 *
+
使用备份期间指定的相同分块系数。

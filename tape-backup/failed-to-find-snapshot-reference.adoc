= 找不到快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Failed to find snapshot`

* * 发生原因 *
+
为备份指定的快照不可用。

* * 更正操作 *
+
检查指定的快照是否可用。如果没有、请使用正确的快照重试。

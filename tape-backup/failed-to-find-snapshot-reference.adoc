= 找不到快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`找不到 snapshot`

* * 发生原因 *
+
为备份指定的 Snapshot 副本不可用。

* * 更正操作 *
+
检查指定的 Snapshot 副本是否可用。如果不是，请使用正确的 Snapshot 副本重试。

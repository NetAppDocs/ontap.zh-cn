= 磁带读取时出现介质错误
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`m磁带读取时出现介质错误`

* * 发生原因 *
+
要从中还原数据的磁带已损坏，并且可能不包含完整的备份数据。

* * 更正操作 *
+
如果您确定磁带具有完整的备份，请重试还原操作。如果磁带不包含完整备份，则无法执行还原操作。

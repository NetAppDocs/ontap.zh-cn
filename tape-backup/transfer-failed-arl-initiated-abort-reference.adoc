= 传输失败（ ARL 已启动中止）
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`传输失败（ ARL 已启动中止）`

* * 发生原因 *
+
在 SMTape 操作进行期间，如果启动了聚合重新定位，则 SMTape 操作将中止。

* * 更正操作 *
+
在聚合重新定位操作完成后执行 SMTape 操作。

= 卷路径无效
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`卷路径无效`

* * 发生原因 *
+
未找到用于备份或还原操作的指定卷。

* * 更正操作 *
+
使用有效的卷路径和卷名称重试此作业。

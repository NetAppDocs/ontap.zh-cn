= 已在磁带末尾
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`已在磁带末尾`

* * 发生原因 *
+
磁带不包含任何数据，或者必须重新卷绕。

* * 更正操作 *
+
如果磁带不包含数据，请使用包含备份的磁带并重试还原作业。否则，倒带并重试还原作业。

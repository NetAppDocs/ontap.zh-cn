= 无法锁定快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Failed to lock snapshot`

* * 发生原因 *
+
快照正在使用中或已被删除。

* * 更正操作 *
+
如果快照正在由其他操作使用、请等待此操作完成、然后重试备份。如果快照已删除、则无法执行备份。

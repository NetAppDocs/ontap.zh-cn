= 由于 Snapshot 自动删除，作业已中止
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Job aborted due to Snapshot autodelete`

* * 发生原因 *
+
此卷没有足够的空间，并已触发 Snapshot 副本自动删除。

* * 更正操作 *
+
释放卷中的空间，然后重试此作业。

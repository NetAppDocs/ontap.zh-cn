= ndmpd 会话 session_ID 未处于活动状态
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`ndmpd 会话 session_ID 未处于活动状态`

* * 发生原因 *
+
NDMP 会话可能不存在。

* * 更正操作 *
+
使用 `ndmpd status` 命令查看活动的 NDMP 会话。

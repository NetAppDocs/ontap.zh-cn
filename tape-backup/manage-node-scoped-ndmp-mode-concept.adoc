= 管理 FlexVol 卷的节点范围的 NDMP 模式概述
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 NDMP 选项和命令在节点级别管理 NDMP 。您可以使用 `options` 命令修改 NDMP 选项。要执行磁带备份和还原操作，您必须使用特定于 NDMP 的凭据来访问存储系统。

有关 `options` 命令的详细信息，请参见手册页。

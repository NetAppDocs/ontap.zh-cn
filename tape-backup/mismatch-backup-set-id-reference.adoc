= 备份集 ID 不匹配
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * 消息 *
+
`m备份集 ID 不匹配`

* * 发生原因 *
+
磁带更改期间加载的磁带不属于备份集。

* * 更正操作 *
+
加载正确的磁带并重试此作业。

= 什么是日志记录事件
:icons: font
:imagesdir: ../media/


[role="lead"]
以日志开头的消息的事件字段指定日志记录的开始或日志记录的结束。

它包含下表所示的一个事件：

|===
| 事件 | Description 


 a| 
start_Logging
 a| 
指示日志记录的开始或日志记录在禁用后已重新打开。



 a| 
stop_Logging
 a| 
指示日志记录已关闭。

|===

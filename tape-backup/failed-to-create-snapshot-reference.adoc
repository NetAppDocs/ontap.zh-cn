= 无法创建快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Failed to create snapshot`

* * 发生原因 *
+
卷包含的快照数已达到上限。

* * 更正操作 *
+
删除一些快照、然后重试备份操作。

= 无法创建快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * 消息 *
+
`Failed to create snapshot`

* * 发生原因 *
+
此卷已包含最大 Snapshot 副本数。

* * 更正操作 *
+
删除一些 Snapshot 副本，然后重试备份操作。

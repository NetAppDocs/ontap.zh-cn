= 创建嵌套的 igroup
:toc: macro
:hardbreaks:
:toclevels: 1

:toclevels: 1
:nofooter: 
:icons: font
:linkattrs:

[role="lead"]
从 ONTAP 9.1.1 开始，您可以创建包含其他现有 igroup 的 igroup 。

. 在 System Manager 中，单击 * 主机 > SAN 启动程序组 * ，然后单击 * 添加 * 。
. 输入 igroup * 名称 * 和 * 问题描述 * 。
+
问题描述用作 igroup 别名。

. 选择 * Storage VM* 和 * 主机操作系统 * 。
+

NOTE: 创建嵌套 igroup 后，无法更改此 igroup 的操作系统类型。

. 在 * 启动程序组成员 * 下，选择 * 现有启动程序组 * 。
+
您可以使用 * 搜索 * 来查找并选择要添加的启动程序组。

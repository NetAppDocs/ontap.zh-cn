= 配置 SVM 范围的 NDMP 概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果 DMA 支持集群感知型备份（ Cluster Aware Backup ， CAB ）扩展，则可以通过启用 SVM 范围的 NDMP ，配置备份用户帐户以及为数据和控制连接配置 LIF 来备份集群中不同节点托管的所有卷。

DMA 必须支持 CAB 扩展。

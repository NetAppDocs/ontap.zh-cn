= 在 SVM 级别或节点级别配置 NDMP 概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果备份应用程序支持集群感知型备份（ Cluster Aware Backup ， CAB ），则可以在集群（管理 SVM ）级别将 NDMP 配置为 SVM 范围，从而可以备份跨集群中不同节点托管的所有卷。否则，您可以配置节点范围的 NDMP ，以便备份该节点上托管的所有卷。

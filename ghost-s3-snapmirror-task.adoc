= 使用 S3 SnapMirror 备份 ONTAP S3 服务器
:icons: font
:imagesdir: ../media/


从 ONTAP 9.10.1 开始，您可以使用熟悉的 SnapMirror 镜像和备份功能来保护 ONTAP S3 对象存储中的存储分段。此外，与标准 SnapMirror 不同， S3 SnapMirror 可以具有非 NetApp 目标。阅读有关此主题的更多信息 xref:s3-snapmirror/index.adoc[此处]。

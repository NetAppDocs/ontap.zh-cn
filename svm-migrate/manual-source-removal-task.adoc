= 客户端切换后手动移除源ONTAP SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果在禁用源清理的情况下执行 SVM 迁移，则可以在客户端转换完成后手动删除源 SVM 。

.步骤
. 确认状态为已准备好进行源清理：
+
[source, cli]
----
vserver migrate show
----
. 清理源：
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----

= 监控卷迁移
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
除了使用`vserver migrate show`命令监控SVM的整体迁移之外、您还可以监控SVM包含的卷的迁移状态。

.步骤
. 检查卷迁移状态：
+
`dest_clust> vserver migrate show-volume`

= 通过添加许可证密钥启用新功能
:toc: macro
:hardbreaks:
:toclevels: 1

[role="lead"]
某些 ONTAP 功能可通过许可证密钥启用。您可以使用 ONTAP 系统管理器添加许可证密钥。

.步骤
. 单击 * 集群 > 设置 * 。
. 在 * 许可证 * 下，单击 image:icon_arrow.gif["箭头图标"]。
. 单击 * 添加 * 。

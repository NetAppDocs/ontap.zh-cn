= 本地用户密码的要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
默认情况下，本地用户密码必须满足复杂性要求。密码复杂度要求与 Microsoft Windows _local security policy_ 中定义的要求类似。

密码必须满足以下条件：

* 长度必须至少为六个字符
* 不得包含用户帐户名称
* 必须包含以下四个类别中至少三个类别的字符：
+
** 大写英文字符（ A 到 Z ）
** 小写英文字符（ a 到 z ）
** 基数为 10 位（ 0 到 9 ）
** 特殊字符：
+
~@ # $ % ｛ caret ｝ & * _ - + = ` \ | （） [ ] ：； " < > ， .？/

= 使用命令行界面设置文件和文件夹安全性的用例
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
由于您可以在本地应用和管理文件和文件夹安全性，而无需远程客户端的参与，因此可以显著缩短为大量文件或文件夹设置批量安全性所需的时间。

在以下使用情形中，使用命令行界面设置文件和文件夹安全性会很有用：

* 在大型企业环境中存储文件，例如主目录中的文件存储
* 数据迁移
* 更改 Windows 域
* 跨 NTFS 文件系统实现文件安全和审核策略标准化

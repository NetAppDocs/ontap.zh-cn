= 在 NAS 命名空间中创建和管理数据卷概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要在 NAS 环境中管理文件访问，您必须管理 Storage Virtual Machine （ SVM ）上的数据卷和接合点。其中包括规划命名空间架构，创建具有或不具有接合点的卷，挂载或卸载卷以及显示有关数据卷和 NFS 服务器或 CIFS 服务器命名空间的信息。

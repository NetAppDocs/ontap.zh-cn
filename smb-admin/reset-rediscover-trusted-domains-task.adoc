= 重置和重新发现受信任域
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以强制重新发现所有受信任域。当受信任域服务器未正确响应或信任关系发生更改时，此功能非常有用。只会发现与主域具有双向信任的域，即包含 CIFS 服务器计算机帐户的域。

.步骤
. 使用 `vserver cifs domain trusts rediscover` 命令重置和重新发现受信任域。
+
`vserver cifs domain trusts rediscover -vserver vs1`

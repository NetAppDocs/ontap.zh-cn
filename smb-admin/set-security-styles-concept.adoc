= 设置安全模式的位置和时间
:icons: font
:imagesdir: ../media/


[role="lead"]
可以在 FlexVol 卷（根卷或数据卷）和 qtree 上设置安全模式。安全模式可以在创建时手动设置，自动继承或稍后更改。

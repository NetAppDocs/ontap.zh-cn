= SMB 服务器支持概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 Storage Virtual Machine （ SVM ）上启用和配置 SMB 服务器，以使 SMB 客户端能够访问集群上的文件。

* 集群中的每个数据 SVM 只能绑定到一个 Active Directory 域。
* 数据 SVM 不需要绑定到同一个域。
* 多个 SVM 可以绑定到同一个域。


在创建 SMB 服务器之前，您必须配置用于提供数据的 SVM 和 LIF 。如果您的数据网络不平整，则可能还需要配置 IP 空间，广播域和子网。网络管理指南 _ 包含详细信息。

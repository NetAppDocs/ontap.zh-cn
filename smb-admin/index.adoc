= 了解ONTAP SMB
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMB 协议提供了 ONTAP 文件访问功能。您可以启用 CIFS 服务器，创建共享和启用 Microsoft 服务。

[NOTE]
====
_SMB_ （服务器消息块）是指通用 Internet 文件系统（ CIFS ）协议的现代方言。您仍会在 ONTAP 命令行界面（ CLI ）和 OnCommand 管理工具中看到 _CIFS_ 。

====

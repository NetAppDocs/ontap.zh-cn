= 了解如何为ONTAP SMB服务器禁用IPv6
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
即使使用网络选项在集群上启用了 IPv6 ，您也不能使用同一命令为 SMB 禁用 IPv6 。而是在集群管理员禁用集群上最后一个启用了 IPv6 的接口时， ONTAP 会禁用 IPv6 。您应与集群管理员就启用了 IPv6 的接口的管理事宜进行沟通。

有关在集群上禁用 IPv6 的详细信息，请参见 _Network Management Guide_ 。

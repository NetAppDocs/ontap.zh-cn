= 分配权限
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以直接为本地用户或域用户分配权限。或者，您也可以将用户分配给已分配权限与这些用户所需功能匹配的本地组。

* 您可以为创建的组分配一组权限。
+
然后，将用户添加到具有所需权限的组。

* 您还可以将本地用户和域用户分配给默认权限与要授予这些用户的权限匹配的预定义组。

= 了解如何使用ONTAP SMB签名增强网络安全性
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMB 签名有助于确保 SMB 服务器和客户端之间的网络流量不会受到影响；它可以通过防止重放攻击来实现这一点。默认情况下，当客户端请求 SMB 签名时， ONTAP 支持 SMB 签名。或者，存储管理员可以将 SMB 服务器配置为需要 SMB 签名。

= 配置 BranchCache 概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 ONTAP 命令在 SMB 服务器上配置 BranchCache 。要实施 BranchCache ，还必须在要缓存内容的分支机构配置客户端以及托管缓存服务器（可选）。

如果您将 BranchCache 配置为在共享基础上启用缓存，则必须在要提供 BranchCache 缓存服务的 SMB 共享上启用 BranchCache 。

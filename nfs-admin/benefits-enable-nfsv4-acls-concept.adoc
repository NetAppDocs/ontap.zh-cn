= 了解为 ONTAP SVM 启用 NFSv4 ACL 的好处
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
启用 NFSv4 ACL 具有许多优势。

启用 NFSv4 ACL 的优势包括：

* 更精细地控制用户对文件和目录的访问
* 提高 NFS 安全性
* 改进了与 CIFS 的互操作性
* 取消了每个用户 16 个组的 NFS 限制

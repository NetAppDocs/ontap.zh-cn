= FlexVol 卷的导出策略限制和嵌套接合
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您将导出策略配置为在嵌套接合上设置限制性较低的策略，而在更高级别的接合上设置限制性较强的策略，则对较低级别的接合的访问可能会失败。

您应确保较高级别的接合与较低级别的接合相比具有较少限制的导出策略。

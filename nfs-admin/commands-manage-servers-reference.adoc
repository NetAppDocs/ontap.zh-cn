= 用于管理 NFS 服务器的命令
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用特定的 ONTAP 命令来管理 NFS 服务器。

[cols="35,65"]
|===


| 如果您要 ... | 使用此命令 ... 


 a| 
创建 NFS 服务器
 a| 
`Vserver nfs create`



 a| 
显示 NFS 服务器
 a| 
`vserver nfs show`



 a| 
修改 NFS 服务器
 a| 
`Vserver nfs modify`



 a| 
删除 NFS 服务器
 a| 
`Vserver nfs delete`



 a| 
隐藏 NFSv3 挂载点下的 ` .snapshot` 目录列表

[NOTE]
====
即使启用了选项，仍允许显式访问 ` 。 snapshot` 目录。

==== a| 
`vserver nfs` 命令并启用 ` -v3-hide-snapshot` 选项

|===
有关详细信息，请参见每个命令的手册页。

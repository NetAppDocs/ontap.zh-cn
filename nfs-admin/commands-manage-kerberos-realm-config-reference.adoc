= 用于管理 NFS Kerberos 域配置的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用特定的 ONTAP 命令来管理 NFS Kerberos 域配置。

[cols="35,65"]
|===
| 如果您要 ... | 使用此命令 ... 


 a| 
创建 NFS Kerberos 域配置
 a| 
`vserver nfs kerberos realm create`



 a| 
显示 NFS Kerberos 域配置
 a| 
`vserver nfs kerberos realm show`



 a| 
修改 NFS Kerberos 域配置
 a| 
`vserver nfs kerberos realm modify`



 a| 
删除 NFS Kerberos 域配置
 a| 
`vserver nfs kerberos realm delete`

|===
有关详细信息，请参见每个命令的手册页。

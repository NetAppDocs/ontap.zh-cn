= 启用或禁用 NFSv3
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过修改 ` -v3` 选项来启用或禁用 NFSv3 。这样，客户端就可以使用 NFSv3 协议访问文件。默认情况下， NFSv3 处于启用状态。

.步骤
. 执行以下操作之一：
+
[cols="20,80"]
|===


| 如果您要 ... | 输入命令 ... 


 a| 
启用 NFSv3 ：
 a| 
`vserver nfs modify -vserver vserver_name -v3 enabled`



 a| 
禁用 NFSv3
 a| 
`vserver nfs modify -vserver vserver_name -v3 disabled`

|===

= 了解 ONTAP NAS SVM 基于身份验证的限制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
通过基于身份验证的限制，您可以指定哪些客户端计算机以及哪些用户可以连接到 Storage Virtual Machine （ SVM ）。

ONTAP 支持从 UNIX 和 Windows 服务器进行 Kerberos 身份验证。

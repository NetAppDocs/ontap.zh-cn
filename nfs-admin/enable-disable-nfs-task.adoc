= 为 ONTAP SVM 启用或禁用 NFS
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 或 CLI 启用或禁用 NFS 版本。这允许使用 NFS 协议的客户端进行文件访问。

默认情况下，NFSv3 在所有支持的 ONTAP 9 版本中均处于启用状态。从 ONTAP 9.9.1 开始，NFSv4.1 和 NFSv4.2 默认处于启用状态；在早期版本中，默认情况下它们处于禁用状态。

从 ONTAP 9.19.1 开始，您可以独立启用或禁用 NFSv4.1 和 NFSv4.2。在 ONTAP 9.19.1 之前，只能同时启用或禁用 NFSv4.1 和 NFSv4.2。默认情况下，除非管理员明确禁用，否则在创建新 NFS 服务时，NFSv4.1 和 NFSv4.2 均已启用。


NOTE: 不支持对 NFSv4.2 文件访问事件的 FPolicy 监视。如果配置 FPolicy 监控，则需要在 SVM 上禁用 NFSv4.2。有关参考信息，请参阅 link:../nas-audit/supported-file-operation-filter-fpolicy-nfsv4-concept.html["ONTAP FPolicy 为 NFSv4 监控的支持的文件操作和过滤器组合"]。

[role="tabbed-block"]
====
.System Manager
--
.步骤
. 在 System Manager 中，选择 *Cluster > Storage VMs*，选择存储 VM，选择 *Settings*，然后选择 image:icon_gear.gif["操作图标"] 或 *NFS* 下的 image:icon-edit-pencil-blue-outline.png["编辑图标"]。
. 选中 *Enable NFS* 复选框。
. 启用或禁用 NFS 协议：
+
** 要启用 NFSv3，请选中 *Enable support version 3* 复选框并考虑以下选项：
+
*** 选中 *Enable 64-bit FSIDs and file identifiers* 复选框以启用对 NFSv3 的 64 位文件系统 ID 和文件标识符的支持。


** 要禁用 NFSv3，请清除 *Enable support version 3* 复选框。
** 要启用 NFSv4.0，请选中 *Enable support version 4* 复选框并考虑以下选项：
+
*** 选中 *Enable ACL* 复选框以允许使用访问控制列表。
*** 选中 *Enable read delegation* 复选框以允许只读访问。
*** 选中 *Enable write delegation* 复选框以允许写入访问。
*** 选中 *启用 64 位 FSID 和文件标识符* 复选框以启用对 64 位文件系统 ID 和文件标识符的支持。此复选框适用于 NFSv4.0、NFSv4.1 和 NFSv4.2。这与用于控制对 NFSv3 的 64 位支持的复选框分开。


** 要禁用 NFSv4.0，请清除 *Enable support version 4* 复选框。
** 要启用 NFSv4.1，请选中 *Enable support version 4.1 and 4.2* 复选框，然后选中 *Enable support version 4.1* 复选框。考虑以下选项：
+
*** 选中 *Enable ACL* 复选框以允许使用访问控制列表。
*** 选中 *Enable read delegation* 复选框以允许只读访问。
*** 选中 *Enable write delegation* 复选框以允许写入访问。
*** 选中 *pNFS* 复选框以允许客户端使用并行 NFS (pNFS) 以提高性能。
*** 选中 *启用 64 位 FSID 和文件标识符* 复选框以启用对 64 位文件系统 ID 和文件标识符的支持。此复选框适用于 NFSv4.0、NFSv4.1 和 NFSv4.2。这与用于控制对 NFSv3 的 64 位支持的复选框分开。


** 要禁用 NFSv4.1，请清除 *Enable support version 4.1* 复选框。
** 要启用 NFSv4.2，请选中 *Enable support version 4.1 and 4.2* 复选框，然后选中 *Enable support version 4.2* 复选框。考虑以下选项：
+
*** 选中 *Enable ACL* 复选框以允许使用访问控制列表。
*** 选中 *Enable read delegation* 复选框以允许只读访问。
*** 选中 *Enable write delegation* 复选框以允许写入访问。
*** 选中 *pNFS* 复选框以允许客户端使用并行 NFS (pNFS) 以提高性能。
*** 选中 *启用 64 位 FSID 和文件标识符* 复选框以启用对 64 位文件系统 ID 和文件标识符的支持。此复选框适用于 NFSv4.0、NFSv4.1 和 NFSv4.2。这与用于控制对 NFSv3 的 64 位支持的复选框分开。


** 要禁用 NFSv4.2，请清除 *Enable support version 4.2* 复选框。


. 或者，您可以为 NFS 协议指定 *V4 ID 域名*。
. 选择 * 保存 * 。

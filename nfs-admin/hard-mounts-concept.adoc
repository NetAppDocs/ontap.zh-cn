= 使用硬挂载
:icons: font
:imagesdir: ../media/


[role="lead"]
在排除挂载问题时，您需要确保使用的挂载类型正确。NFS 支持两种挂载类型：软挂载和硬挂载。出于可靠性考虑，您应仅使用硬挂载。

您不应使用软挂载，尤其是在可能频繁出现 NFS 超时的情况下。这些超时可能会导致出现争用情况，进而导致数据损坏。

= NFS 参考概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 包括可用于 NFS 协议的文件访问功能。您可以启用 NFS 服务器并导出卷或 qtree 。

您可以在以下情况下执行这些操作步骤：

* 您希望了解 ONTAP NFS 协议功能的范围。
* 您希望执行不太常见的配置和维护任务，而不是基本 NFS 配置。
* 您希望使用命令行界面（ CLI ），而不是 System Manager 或自动化脚本编写工具。

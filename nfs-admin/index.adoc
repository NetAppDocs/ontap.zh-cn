= NFS 参考概述
:icons: font
:imagesdir: ../media/


[role="lead"]
此内容介绍了可用于 NFS 协议的 ONTAP 文件访问功能。它介绍了如何启用 NFS 服务器以及如何导出卷或 qtree 。其中包括规划指导和详细的实施说明。

在以下情况下，应使用此内容：

* 您希望了解 ONTAP NFS 协议功能的范围。
* 您希望执行不太常见的配置和维护任务，而不是基本 NFS 配置。
* 您希望使用命令行界面（ CLI ），而不是 ONTAP 系统管理器或自动脚本编写工具。

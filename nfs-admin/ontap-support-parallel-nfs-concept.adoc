= 支持并行 NFS 的 ONTAP
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 支持并行 NFS （ pNFS ）。pNFS 协议可使客户端直接访问分布在集群多个节点上的一组文件的数据，从而提高了性能。它可以帮助客户端找到卷的最佳路径。

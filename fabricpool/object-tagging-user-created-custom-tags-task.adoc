= 使用用户创建的自定义标记概述进行对象标记
:icons: font
:imagesdir: ../media/


[role="lead"]
从 ONTAP 9.8 开始， FabricPool 支持使用用户创建的自定义标记进行对象标记，以便对对象进行分类和排序，从而简化管理。如果您是具有管理员权限级别的用户，则可以创建新的对象标记，以及修改，删除和查看现有标记。

= 管理 FabricPool 概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
为了帮助您满足存储分层需求，您可以使用 ONTAP 显示卷中处于非活动状态的数据量，向 FabricPool 添加或移动卷，监控 FabricPool 的空间利用率，或者修改卷的分层策略或分层最小冷却期。

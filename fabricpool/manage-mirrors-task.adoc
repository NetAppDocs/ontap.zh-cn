= 了解ONTAP FabricPool镜像
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
为了确保在发生灾难时可以访问数据存储中的数据，并使您能够替换数据存储，您可以通过添加第二个数据存储来配置 FabricPool 镜像，以将数据同步分层到两个数据存储中。您可以向新的或现有的 FabricPool 配置添加第二个数据存储，监控镜像状态，显示 FabricPool 镜像详细信息，提升镜像以及删除镜像。您必须运行ONTAP 9.7或更高版本。

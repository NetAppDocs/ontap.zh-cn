= 触发计划内迁移和分层
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您不希望等待默认分层扫描，则可以随时触发分层扫描请求。

.步骤
. 使用带有 `trigger` 选项的 `volume object-store` 命令请求迁移和分层。
+
[listing]
----
volume object-store tiering trigger [ -vserver <vserver name> ] *VServer Name [-volume] <volume name> *Volume Name
----

= 查看卷上的现有标记
:icons: font
:imagesdir: ../media/


[role="lead"]
在向列表附加新标记之前，您可以查看卷上的现有标记，以查看哪些标记可用。

.步骤
. 使用带有 ` -tiering-object-tags` 选项的 `volume show` 命令查看卷上的现有标记。
+
[listing]
----
volume show [ -vserver <vserver name> ] -volume <volume_name> -fields -tiering-object-tags
----

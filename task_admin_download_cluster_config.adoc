= 下载集群配置
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
从ONTAP 9.11.1开始、您可以使用System Manager下载集群配置。

.步骤
. 单击 * 集群 > 概述 * 。
. 单击 image:icon-more-kebab-blue-bg.gif["更多选项图标"] 以显示下拉菜单。
. 选择*下载配置*。
. 选择HA对，然后单击*Download*。
+
此配置将下载为Excel电子表格。

+
** 第一个工作表包含集群详细信息。
** 其他工作表包含节点详细信息。

= 向本地层添加容量的工作流(扩展聚合)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要向本地层添加容量(扩展聚合)、您必须先确定要添加到的本地层、确定需要多少新存储、安装新磁盘、分配磁盘所有权、并根据需要创建新的RAID组。

您可以使用System Manager或命令行界面添加容量。

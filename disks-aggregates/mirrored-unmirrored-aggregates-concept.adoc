= 已镜像和未镜像聚合
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用名为 _raid_ 的可选功能同步镜像存储在不同 SyncMirror 组中的副本或 _plexes_ 中的聚合数据。丛可确保在出现故障的磁盘数量超过 RAID 类型所能保护的磁盘数量或与 RAID 组磁盘的连接断开时防止数据丢失。

= 删除本地层(聚合)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果本地层上没有卷、则可以删除本地层(聚合)。

`storage aggregate delete`命令可删除存储聚合。如果聚合上存在卷、则此命令将失败。如果聚合附加了对象存储、则除了删除聚合之外、此命令还会删除对象存储中的对象。在此命令中、不会更改对象存储配置。

以下示例将删除名为"`aggr1`"的聚合：

....
> storage aggregate delete -aggregate aggr1
....

= 确定聚合上的卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在对聚合执行操作之前，您可能需要确定哪些卷驻留在聚合上，例如重新定位聚合或使其脱机。

.步骤
. 要显示位于聚合上的卷，请输入
+
`volume show -aggregate aggregate_name`

+
此时将显示位于指定聚合上的所有卷。

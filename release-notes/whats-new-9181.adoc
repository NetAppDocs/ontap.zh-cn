== 网络

[cols="30%,70%"]
|===
| 更新 | Description 


 a| 
xref:../networking/ipsec-prepare.html[IPsec 硬件卸载 IPv6 支持]
 a| 
IPsec硬件卸载支持已扩展到IPv6。



 a| 
xref:../networking/configure_network_security_using_federal_information_processing_standards_fips.html[OpenSSL PQC 算法]
 a| 
ONTAP支持用于 SSL 的后量子计算加密算法。这些算法可针对未来潜在的量子计算攻击提供额外保护，并且在 SSL FIPS 模式禁用时可用。



 a| 
xref:../networking/configure-backend-encryption.html[ONTAP后端集群网络加密]
 a| 
您可以为ONTAP后端集群网络上的传输中数据启用 TLS 加密。



 a| 
xref:../high-availability/ha-pair-encryption.html[ONTAP HA 流量网络加密]
 a| 
您可以为高可用性 (HA) 对中的节点之间的流量启用加密。

|===


== SAN

[cols="30%,70%"]
|===
| 更新 | Description 


 a| 
xref:../san-admin/nvme-copy-offload.html[NVMe 复制卸载]
 a| 
NVMe 复制卸载功能使 NVMe 主机能够将复制操作从其 CPU 卸载到ONTAP存储控制器的 CPU。主机可以将数据从一个 NVMe 命名空间复制到另一个 NVMe 命名空间，同时将其 CPU 资源保留给应用程序工作负载。

|===

== 安全性

[cols="30%,70%"]
|===
| 更新 | Description 


 a| 
xref:../anti-ransomware/use-cases-restrictions-concept.html#supported-configurations[FlexGroup卷支持 ARP/AI]
 a| 
NetApp FlexGroup卷支持 ARP/AI，提供基于机器学习的高级勒索软件检测和保护。在ONTAP 9.17.1 及更早版本中， FlexGroup卷仅支持第一代 ARP。从ONTAP 9.18.1 开始， FlexGroup卷在本地ONTAP平台（AFF和FAS）和虚拟ONTAP部署（包括Cloud Volumes ONTAP和ONTAP Select ）上与FlexVol卷具有同等功能。因此，ARP/AI 成为FlexVol和FlexGroup卷的默认 ARP 技术。

|===

= 管理目录和文件
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
展开 System Manager 卷显示以查看和删除目录和文件。

从 ONTAP 9.1.1 开始，目录会通过低延迟快速目录删除功能进行删除。

有关在 ONTAP 9.9.1 及更高版本中查看文件系统的详细信息，请参见 link:concept_nas_file_system_analytics_overview.html["文件系统分析概述"]。

.步骤
. 选择 * 存储 > 卷 * 。展开卷以查看其内容。

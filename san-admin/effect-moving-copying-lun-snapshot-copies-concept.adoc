= 移动或复制 LUN 对 Snapshot 副本的影响概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snapshot 副本在卷级别创建。如果将 LUN 复制或移动到其他卷，则目标卷的 Snapshot 副本策略将应用于复制或移动的卷。如果未为目标卷创建 Snapshot 副本，则不会为移动或复制的 LUN 创建 Snapshot 副本。

= 在 SAN 环境中移动卷的要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
移动包含 LUN 或命名空间的卷之前，必须满足特定要求。

* 对于包含一个或多个 LUN 的卷，每个连接到集群中每个节点的 LUN （ LIF ）至少应具有两个路径。
+
这样可以消除单点故障，并使系统能够承受组件故障。

* 对于包含命名空间的卷，集群必须运行 ONTAP 9.6 或更高版本。
+
运行 ONTAP 9.5 的 NVMe 配置不支持卷移动。

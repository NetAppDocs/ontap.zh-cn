= 从 SVM 中删除 NVMe 服务
:icons: font
:imagesdir: ../media/


[role="lead"]
如果需要，您可以从 Storage Virtual Machine （ SVM ）中删除 NVMe 服务。

.步骤
. 将权限设置更改为高级：
+
`set -privilege advanced`

. 停止 SVM 上的 NVMe 服务：
+
`vserver nvme modify -status -admin down`

. 删除 NVMe 服务：
+
`Vserver nvme delete`

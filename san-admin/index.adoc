= 使用 CLI 进行 SAN 管理概述
:icons: font
:imagesdir: ../media/


[role="lead"]
请按照以下步骤执行基本 SAN 主机配置。您可以使用 iSCSI 和 FC 协议配置和管理 LUN ， igroup 和目标，以及使用 NVMe/FC 协议的命名空间和子系统。

在以下情况下，应使用这些过程：

* 您希望了解 ONTAP SAN 主机配置功能的范围。
* 您希望执行不太常见的配置和维护任务，而不是基本 SAN 配置。
* 您希望使用命令行界面（ CLI ），而不是 System Manager 或自动化脚本编写工具。

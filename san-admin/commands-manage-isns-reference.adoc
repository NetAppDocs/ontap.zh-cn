= 用于管理 iSNS 的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 提供了用于管理 iSNS 服务的命令。

[cols="2*"]
|===
| 如果您要 ... | 使用此命令 ... 


 a| 
配置 iSNS 服务
 a| 
`Vserver iscsi isns create`



 a| 
启动 iSNS 服务
 a| 
`SVM iSCSI iSNS start`



 a| 
修改 iSNS 服务
 a| 
`SVM iSCSI iSNS modify`



 a| 
显示 iSNS 服务配置
 a| 
`vserver iscsi isns show`



 a| 
强制更新已注册的 iSNS 信息
 a| 
`SVM iSCSI iSNS 更新`



 a| 
停止 iSNS 服务
 a| 
`SVM iSCSI iSNS stop`



 a| 
删除 iSNS 服务
 a| 
`SVM iSCSI iSNS delete`



 a| 
查看命令的手册页
 a| 
`man _command name_`

|===
有关详细信息，请参见每个命令的手册页。

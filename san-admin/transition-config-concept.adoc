= 过渡 SAN 配置的注意事项
:icons: font
:imagesdir: ../media/


[role="lead"]
在 SAN 环境中，在将 7- 模式卷过渡到 ONTAP 期间，需要中断服务。要完成过渡，您需要关闭主机。过渡后，必须更新主机配置，然后才能开始在 ONTAP 中提供数据

您需要计划一个维护窗口，在此期间可以关闭主机并完成过渡。

link:https://docs.netapp.com/us-en/ontap-7mode-transition/copy-based/index.html["基于副本的过渡"]

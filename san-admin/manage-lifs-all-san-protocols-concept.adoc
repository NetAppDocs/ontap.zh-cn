= 了解适用于 SAN 环境的 ONTAP LIF
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 系统中的 LUN 和命名空间包含在 Storage Virtual Machine (SVM) 中。网络接口 (LIF) 允许您的 SAN 客户端连接到 SVM 并访问其包含的 LUN 和命名空间中的数据。

* 您应该为每个支持 SAN 的 Storage Virtual Machine (SVM) 配置一个管理 LIF。
* 您可以为每个节点配置两个 LIF，一个用于与 FC 一起使用的每个交换矩阵，另一个用于 iSCSI 的单独以太网网络。
* 支持直接连接或使用以太网交换机进行连接。必须为两种类型的连接配置 LIF。

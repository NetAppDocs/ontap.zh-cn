= iSCSI 的 TCP 端口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
iSCSI 协议在 ONTAP 中配置为使用 TCP 端口号 3260 。

ONTAP 不支持更改 iSCSI 的端口号。端口号 3260 已注册为 iSCSI 规范的一部分，不能由任何其他应用程序或服务使用。

= EMS 配置工作流
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须将重要的 EMS 事件通知配置为通过电子邮件发送，转发到系统日志服务器，转发到 SNMP 陷阱主机或转发到 REST API 服务器。这有助于您及时采取更正操作，避免系统中断。

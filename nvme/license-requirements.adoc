= NVMe许可证要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
从 ONTAP 9.5 开始，需要许可证才能支持 NVMe 。如果在 ONTAP 9.4 中启用了 NVMe ，则在升级到 ONTAP 9.5 之后，可以在 90 天的宽限期内获取许可证。

您可以使用以下命令启用许可证：

`ssystem license add -license-code _nva_license_key_`

= 了解 ONTAP Vscan 的防病毒配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vscan是NetApp开发的防病毒扫描解决方案、支持客户保护其数据免受病毒或其他恶意代码的危害。

当客户端通过SMB访问文件时、Vscan会执行病毒扫描。您可以将Vscan配置为按需或按计划进行扫描。您可以使用ONTAP命令行界面(CLI)或ONTAP应用程序编程接口(API)与Vscan进行交互。

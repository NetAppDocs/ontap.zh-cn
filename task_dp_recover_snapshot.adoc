= 从 Snapshot 副本恢复
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以通过从 Snapshot 副本还原来将卷恢复到较早的时间点。

此操作步骤可从 Snapshot 副本还原卷。

.步骤
. 单击 * 存储 * 并选择一个卷。
. 在 * Snapshot 副本 * 下，单击 image:icon_kabob.gif["菜单选项"] 在要还原的 Snapshot 副本旁边，选择 * 还原 * 。

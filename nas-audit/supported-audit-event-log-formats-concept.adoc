= 支持的审核事件日志格式是什么
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
转换后的审核事件日志支持的文件格式为 `EVTX` 和 `XML` 文件格式。

您可以在创建审核配置时指定文件格式的类型。默认情况下、ONTAP会将二进制日志转换为 `EVTX` 文件格式。

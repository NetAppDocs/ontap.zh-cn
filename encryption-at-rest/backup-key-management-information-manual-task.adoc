= 手动备份板载密钥管理信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
配置板载密钥管理器密码短语时，应将板载密钥管理信息复制到存储系统外的安全位置。

.您需要什么？ #8217 ；将需要什么
* 您必须是集群管理员才能执行此任务。
* 此任务需要高级权限。


所有密钥管理信息都会自动备份到集群的复制数据库（ RDB ）。您还应手动备份密钥管理信息，以便在发生灾难时使用。

.步骤
. 更改为高级权限级别：
+
`set -privilege advanced`

. 显示集群的密钥管理备份信息：
+
[cols="40,60"]
|===


| 对于此 ONTAP 版本 ... | 使用此命令 ... 


 a| 
ONTAP 9.6 及更高版本
 a| 
`s安全密钥管理器板载 show-backup`



 a| 
ONTAP 9.5 及更早版本
 a| 
`security key-manager backup show`

|===


有关完整的命令语法，请参见手册页。

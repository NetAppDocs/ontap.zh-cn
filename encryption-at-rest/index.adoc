= 了解ONTAP数据加密
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp 提供了基于软件和基于硬件的加密技术，可确保在存储介质被重新利用，退回，放置在不当位置或被盗时无法读取空闲数据。

* 使用NetApp卷加密(NVE)的基于软件的加密支持一次对一个卷进行数据加密
* 使用NetApp存储加密(NetApp Storage Encryption、NSE)的基于硬件的加密支持在数据写入时对其进行全磁盘加密(FDE)。

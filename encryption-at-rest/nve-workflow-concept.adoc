= NetApp 卷加密工作流
:icons: font
:imagesdir: ../media/


[role="lead"]
必须先配置密钥管理服务，然后才能启用卷加密。您可以对新卷或现有卷启用加密。

image::../media/nve-workflow.gif[NVE 工作流]

您必须先安装 VE 许可证并配置密钥管理服务，然后才能使用 NVE 对数据进行加密。在安装许可证之前，您应先执行此操作 link:luster-version-support-nve-task.html["确定您的 ONTAP 版本是否支持 NVE"]。

= 卷移动
:icons: font
:imagesdir: ../media/


[role="lead"]
在确定存储容量不平衡后，您可以使用 ONTAP 将 FlexVol 卷移动到同一 Storage VM （ SVM ）中的其他本地层（聚合），节点或这两者，以平衡存储容量。使用 Cloud Volumes ONTAP ，您可以将一个或多个卷移动到另一个 Cloud Volumes ONTAP 系统或另一个聚合，以避免出现容量问题。如果系统达到磁盘限制，您可能需要执行此操作。

= 存储效率
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
云中提供了与内部 ONTAP 相同的存储效率功能。Snapshot 副本，重复数据删除，数据压缩，数据缩减，精简配置， 和 FlexClone 数据克隆均可从 NetApp Cloud 产品中获得。

将数据从内部 ONTAP 迁移到云时，会保留现有存储效率。无论您是移动整个数据集，还是仅将冷数据分层到云，您都不会移动未压缩或重复的数据。

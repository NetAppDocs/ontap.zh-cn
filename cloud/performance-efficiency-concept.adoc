= 云中的性能和效率概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您的内部 ONTAP 系统提供了数据效率功能，可让您以更少的物理空间存储更多的数据，并将很少使用的数据分层到更低的存储成本。无论您是使用混合云配置，还是将整个工作负载迁移到云， ONTAP 都可以帮助您最大限度地提高存储性能和效率。

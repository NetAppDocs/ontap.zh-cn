= 防病毒保护
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能会在内部使用集成的防病毒功能来防止数据受到病毒或其他恶意代码的侵害。使用 Cloud Volumes ONTAP 时，云中也提供同样的防病毒保护。

安装在本地服务器上的 ONTAP 防病毒连接器可处理存储系统与防病毒软件之间的通信。对于 Cloud Volumes ONTAP ，您可以在与 ONTAP 位于同一云中的虚拟机上安装 Antivirus Connector 。

= Storage VM
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage VM （ SVM ）为客户端和主机提供数据。与虚拟机管理程序上运行的虚拟机一样， SVM 也是一个逻辑实体，用于抽象化物理资源。

在内部 ONTAP 环境中，您可以使用 SVM 来分隔工作负载。在 Cloud Volumes ONTAP 中，您可以使用多个 SVM ，也可以使用多个 Cloud Volumes ONTAP 实例。

= 删除ONTAP FlexCache关系
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果不再需要 FlexCache 卷，则可以删除 FlexCache 关系和 FlexCache 卷。

.开始之前
如果您启用了 FlexCache 回写，则必须先将其禁用，然后才能删除 FlexCache 卷。请参阅link:../flexcache-writeback/flexcache-writeback-enable-task.html#disable-write-back-on-a-flexcache-volume["在FlexCache卷上禁用回写"]。

.步骤
. 从包含 FlexCache 卷的集群中，使 FlexCache 卷脱机：
+
`volume offline -vserver svm_name -volume volume_name`

. 删除 FlexCache 卷：
+
`volume flexcache delete -vserver svm_name -volume volume_name`

+
FlexCache 关系详细信息将从原始卷和 FlexCache 卷中删除。

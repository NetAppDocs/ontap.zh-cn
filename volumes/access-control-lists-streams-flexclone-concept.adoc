= 访问控制列表如何与 FlexClone 文件和 FlexClone LUN 配合使用
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone 文件和 FlexClone LUN 会继承其父文件和 LUN 的访问控制列表。

如果父文件包含 Windows NT 流，则 FlexClone 文件也会继承流信息。但是，不能克隆包含六个以上流的父文件。

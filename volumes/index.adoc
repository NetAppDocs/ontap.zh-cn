= 使用 CLI 进行逻辑存储管理概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 ONTAP 命令行界面，您可以创建和管理 FlexVol 卷，使用 FlexClone 技术为卷，文件和 LUN 创建高效副本，创建 qtree 和配额以及管理重复数据删除和数据压缩等效率功能。

在以下情况下，应使用这些过程：

* 您希望了解 ONTAP FlexVol 卷功能和存储效率功能的范围。
* 您希望使用命令行界面（ CLI ），而不是 System Manager 或自动化脚本编写工具。

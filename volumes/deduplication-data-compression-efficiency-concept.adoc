= 重复数据删除、数据压缩、数据缩减和存储效率
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以同时或单独运行重复数据删除，数据压缩和数据缩减，以在 FlexVol 卷上实现最佳空间节省。重复数据删除可消除重复的数据块。数据压缩可压缩数据块，以减少所需的物理存储量。数据缩减可在更少的空间中存储更多数据，以提高存储效率。

[NOTE]
====
从 ONTAP 9.2 开始， AFF 卷默认启用所有实时存储效率功能，例如实时重复数据删除和实时压缩。

====

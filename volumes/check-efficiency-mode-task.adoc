= 检查卷效率模式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 AFF 卷上使用 `volume-efficiency-show` 命令检查是否已设置效率并查看当前效率模式。

.步骤
. 检查卷上的效率模式：
+
` * volume efficiency show -vserver <vserver name> -volume <volume name> -fields storage-efficiency-mode*`

= HA 配置如何与 FlexClone 文件和 FlexClone LUN 配合使用
:icons: font
:imagesdir: ../media/


[role="lead"]
HA 配置支持 FlexClone 文件和 FlexClone LUN 操作。

在 HA 对中，在接管或交还操作正在进行时，您不能在配对系统上创建 FlexClone 文件或 FlexClone LUN 。接管或交还操作完成后，配对系统上所有待定的块共享操作将恢复。

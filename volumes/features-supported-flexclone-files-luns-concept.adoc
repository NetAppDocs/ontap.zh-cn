= FlexClone 文件和 FlexClone LUN 支持的功能
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone 文件和 FlexClone LUN 可与不同的 ONTAP 功能配合使用，例如重复数据删除， Snapshot 副本，配额和卷 SnapMirror 。

FlexClone 文件和 FlexClone LUN 支持以下功能：

* 重复数据删除
* Snapshot 副本
* 访问控制列表
* 配额
* FlexClone 卷
* NDMP
* 卷 SnapMirror
* `volume move` 命令
* 空间预留
* HA 配置

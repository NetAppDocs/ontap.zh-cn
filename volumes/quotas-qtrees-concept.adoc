= 配额如何用于 qtree
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以创建以 qtree 作为目标的配额；这些配额称为 _tree quotas_ 。您还可以为特定 qtree 创建用户和组配额。此外， FlexVol 卷的配额有时会由该卷包含的 qtree 继承。

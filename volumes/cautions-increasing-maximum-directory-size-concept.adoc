= 增加 FlexVol 卷的最大目录大小的注意事项
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexVol 卷的默认最大目录大小取决于型号，并针对系统内存大小进行了优化。在增加最大目录大小之前，请联系技术支持。

您可以使用 `volume modify` 命令的 ` -maxdir-size` 选项来增加特定 FlexVol 卷的默认最大目录大小，但这样做可能会影响系统性能。

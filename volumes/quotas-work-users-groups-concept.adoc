= 配额如何与用户和组结合使用概述
:icons: font
:imagesdir: ../media/


[role="lead"]
如果将某个用户或组指定为配额的目标，则该配额施加的限制将应用于该用户或组。但是，某些特殊组和用户的处理方式有所不同。根据您的环境，可以通过不同的方式为用户指定 ID 。

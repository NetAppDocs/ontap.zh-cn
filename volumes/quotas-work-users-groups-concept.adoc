= 概述如何对用户和组使用配额
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以将用户或组指定为配额的目标。定义配额时、需要考虑几个实施差异。

您需要注意的一些差异包括：

* 用户或组
* UNIX或Windows
* 特殊用户和组
* 是否包含多个ID


根据您的环境、还可以通过不同的方式为用户指定ID。

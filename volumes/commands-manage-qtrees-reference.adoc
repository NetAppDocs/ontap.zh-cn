= 用于管理和配置 qtree 的命令
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用特定的 ONTAP 命令管理和配置 qtree 。

[cols="2*"]
|===
| 如果您要 ... | 使用此命令 ... 


 a| 
创建 qtree
 a| 
`volume qtree create`



 a| 
显示经过筛选的 qtree 列表
 a| 
`volume qtree show`



 a| 
删除 qtree
 a| 
`volume qtree delete`

[NOTE]
====
qtree 命令 `volume qtree delete` 将失败，除非 qtree 为空或添加了 ` -force true` 标志。

====


 a| 
修改 qtree 的 UNIX 权限
 a| 
`volume qtree modify -unix-permissions`



 a| 
修改 qtree 的 CIFS 操作锁定设置
 a| 
`卷 qtree 操作锁定`



 a| 
修改 qtree 的安全设置
 a| 
`卷 qtree 安全性`



 a| 
重命名 qtree
 a| 
`卷 qtree 重命名`



 a| 
显示 qtree 的统计信息
 a| 
`卷 qtree 统计信息`



 a| 
重置 qtree 的统计信息
 a| 
`volume qtree statistics -reset`

|===
[NOTE]
====
使用 `volume rehost` 命令可以对针对该卷的其他并发管理操作执行发生原因，以使其失败。

====

= 手动管理卷效率操作概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过手动运行效率操作来管理效率操作在卷上的运行方式。

您还可以根据以下条件控制效率操作的运行方式：

* 是否使用检查点
* 对现有数据或仅对新数据运行效率操作
* 根据需要停止效率操作


您可以使用 `volume efficiency show` 命令 `schedule` 作为的值 `-fields` 选项以查看分配给卷的计划。

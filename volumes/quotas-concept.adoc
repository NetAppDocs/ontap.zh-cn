= 使用配额的优势
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用配额管理和监控FlexVol卷的资源使用情况。

定义配额有几个优势。您可以使用默认配额、显式配额、派生配额和跟踪配额来以最高效的方式管理磁盘使用量。

.限制资源消耗
您可以限制用户、组使用的或qtree中包含的磁盘空间量或文件数。

.跟踪资源使用情况
可以跟踪用户、组或qtree所使用的磁盘空间量或文件数、而不施加限制。

.通知用户
当资源使用量达到特定级别时、可以生成通知。当用户的磁盘或文件使用量太高时、此选项会向用户发出警告。

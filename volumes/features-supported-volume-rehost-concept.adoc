= 不支持卷重新托管的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
某些功能不支持卷重新托管。

以下功能不支持卷重新托管：

* SVM 灾难恢复
* MetroCluster 配置
* SnapLock 卷
* NetApp卷加密(NVE)卷(在ONTAP 9.8之前的版本中)
+
在ONTAP 9.8之前的版本中、您必须先取消对卷的加密、然后才能重新托管它。卷加密密钥取决于 SVM 密钥。如果将卷移动到另一个 SVM ，并且在源或目标 SVM 上启用了多租户密钥配置，则卷和 SVM 密钥将不匹配。

+
从ONTAP 9.8开始、您可以使用NVE重新托管卷。

* FlexGroup 卷
* 克隆卷

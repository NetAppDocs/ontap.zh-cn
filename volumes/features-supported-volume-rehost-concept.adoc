= 不支持卷重新托管的功能
:icons: font
:imagesdir: ../media/


[role="lead"]
某些功能不支持卷重新托管。

以下功能不支持卷重新托管：

* VDR 设置
* MetroCluster 配置
* SnapLock 卷
* VGE volumes
+
卷加密密钥取决于 SVM 密钥。如果将卷移动到另一个 SVM ，并且在源或目标 SVM 上启用了多租户密钥配置，则卷和 SVM 密钥将不匹配。

* FlexGroup 卷
* 克隆卷

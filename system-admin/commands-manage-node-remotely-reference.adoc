:imagesdir: ../media/

[NOTE]
====
使用这些命令关闭节点或重新启动节点可能发生原因会导致节点不正确关闭（也称为 _dirty shutdown_ ），并且无法替代使用 ONTAP `ssystem node halt` 命令正常关闭。

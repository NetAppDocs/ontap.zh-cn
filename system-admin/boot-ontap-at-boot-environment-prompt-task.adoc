= 在启动环境提示符处启动 ONTAP
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在节点的启动环境提示符处启动 ONTAP 的当前版本或备份版本。

.步骤
. 使用 `ssystem node halt` 命令从存储系统提示符访问启动环境提示符。
+
存储系统控制台将显示启动环境提示符。

. 在启动环境提示符处，输入以下命令之一：
+
|===
| 要启动 ... | 输入 ... 


 a| 
ONTAP 的当前版本
 a| 
`boot_ontap`



 a| 
启动设备中的 ONTAP 主映像
 a| 
`boot_primary`



 a| 
启动设备中的 ONTAP 备份映像
 a| 
`boot_backup`

|===
+
如果您不确定要使用哪个映像，则应在第一个实例中使用 `boot_ontap` 。

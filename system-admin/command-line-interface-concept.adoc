= 使用 ONTAP 命令行界面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 命令行界面（ CLI ）提供了一个基于命令的管理界面视图。您可以在存储系统提示符处输入命令，命令结果将以文本形式显示。

命令行界面命令提示符显示为 `cluster_name::>`。

如果您设置了权限级别(即 `-privilege` 的参数 `set` 命令) `advanced`，提示符中包含星号(*)，例如：

`cluster_name::*>`

= 使用 ONTAP 命令行界面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 命令行界面（ CLI ）提供了一个基于命令的管理界面视图。您可以在存储系统提示符处输入命令，命令结果将以文本形式显示。

CLI 命令提示符显示为 `cluster_name ：： >` 。

如果将权限级别（即 `set` 命令的 ` -privilege` 参数）设置为 `advanced` ，则提示符将包含一个星号（ * ），例如：

`cluster_name ：： * >`

= 用于管理 CLI 会话自动超时期限的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
超时值用于指定 CLI 会话在自动终止之前保持空闲多长时间。CLI 超时值为集群范围。也就是说，集群中的每个节点都使用相同的 CLI 超时值。

默认情况下， CLI 会话的自动超时期限为 30 分钟。

您可以使用 `system timeout` 用于管理CLI会话自动超时期限的命令。

|===
| 如果您要 ... | 使用此命令 ... 


 a| 
显示 CLI 会话的自动超时期限
 a| 
`system timeout show`



 a| 
修改 CLI 会话的自动超时期限
 a| 
`system timeout modify`

|===

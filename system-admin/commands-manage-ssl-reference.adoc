= 用于管理SSL的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `security ssl` 用于管理集群或Storage Virtual Machine (SVM)的SSL协议的命令。

|===
| 如果您要 ... | 使用此命令 ... 


 a| 
为集群或 SVM 启用 SSL ，并将数字证书与其关联
 a| 
`security ssl modify`



 a| 
显示集群或 SVM 的 SSL 配置和证书名称
 a| 
`security ssl show`

|===

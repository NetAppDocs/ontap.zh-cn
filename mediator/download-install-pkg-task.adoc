= 下载ONTAP调解器安装包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
下载ONTAP Mediator 安装包并进行安装。

.步骤
. 从 ONTAP Mediator 下载页面下载 ONTAP Mediator 安装包。
+
https://mysupport.netapp.com/site/products/all/details/ontap-mediator/downloads-tab["ONTAP 调解器下载页面"^]

. 请确保已将 Mediator 安装包放置在当前工作目录中：
+
[listing]
----
[root@sdot-r730-0003a-d6 ~]# ls ontap-mediator-1.11.0.tgz
----
+
[listing]
----
ontap-mediator-1.11.0.tgz
----
+

NOTE: 对于ONTAP 调解器1.4及更早版本、安装程序名为 `ontap-mediator`。

+
如果您的系统无法访问互联网，请确保安装程序可以访问所需的软件包。

. 如有需要，请将 Mediator 安装包移动到安装目录。
. 解压缩安装程序包：
+
`tar xvfz ontap-mediator-1.11.0.tgz`

+

----

ontap-mediator-1.11.0/ontap-mediator-1.11.0.tsr
ontap-mediator-1.11.0/ontap-mediator-1.11.0.sig
----

= 正在删除 ONTAP 调解器
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
如果要从 ONTAP 集群中删除现有 ONTAP 调解器配置，可以使用 `snapmirror mediator remove` 命令执行此操作。

.步骤
. 删除 ONTAP 调解器：
+
`snapmirror mediator remove -mediate-address 12.345.678.90 -peer-cluster cluster_xyz`

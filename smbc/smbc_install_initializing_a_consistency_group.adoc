= 初始化一致性组
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
创建一致性组后，必须对其进行初始化。

您必须是集群或 Storage VM 管理员。

您可以从目标集群初始化一致性组。

.步骤
. 登录到目标集群上的 ONTAP 命令行界面并初始化一致性组：
+
`d目标：： >snapmirror initialize -destination-path vs1_dst ： /CG/CG_DST`

. 确认初始化操作已成功完成。状态应为 `InSync` 。
+
`snapmirror show`

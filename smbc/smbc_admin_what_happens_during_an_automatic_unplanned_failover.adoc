= 自动计划外故障转移操作
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
当主集群关闭或隔离时，会发生自动计划外故障转移（ AUTOFO ）操作。  如果发生这种情况，二级集群将转换为主集群，并开始为客户端提供服务。此操作只能在 ONTAP 调解器的协助下执行。


NOTE: 在自动计划外故障转移之后，请务必重新扫描主机 LUN I/O 路径，以确保 I/O 路径不会丢失。

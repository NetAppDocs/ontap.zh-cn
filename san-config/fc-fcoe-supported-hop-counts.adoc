= 支持的最大FC和FCoE跃点计数
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
跃点计数是指启动程序（主机）与目标（存储系统）之间路径中的交换机数量。主机和存储系统之间支持的最大FC跃点计数因交换机供应商而异。

Cisco Systems 的文档还将此值称为 SAN 网络结构的直径。

对于 FCoE ，可以将 FCoE 交换机连接到 FC 交换机。对于端到端 FCoE 连接， FCoE 交换机必须运行支持以太网交换机间链路（ ISL ）的固件版本。

[cols="2*"]
|===
| 交换机供应商 | 支持的跃点计数 


 a| 
Brocade
 a| 
* FC 为 7
* 5 表示 FCoE




 a| 
Cisco
 a| 
* FC 为 7
* 最多 3 个交换机可以是 FCoE 交换机。


|===

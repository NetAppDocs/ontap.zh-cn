= 显示有关 FC 目标适配器的信息
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `network fcp adapter show` 命令显示系统中任何 FC 适配器的系统配置和适配器信息。

.步骤
. 使用 `network fcp adapter show` 命令显示有关 FC 适配器的信息。
+
输出将显示所用每个插槽的系统配置信息和适配器信息。

+
`network fcp adapter show -instance -node node1 -adapter 0a`

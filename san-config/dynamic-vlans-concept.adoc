= 动态 VLANs
:icons: font
:imagesdir: ../media/


[role="lead"]
动态 VLAN 基于 MAC 地址。您可以通过指定要包括的成员的 MAC 地址来定义 VLAN 。

动态 VLAN 具有灵活性，不需要映射到设备物理连接到交换机的物理端口。您可以将缆线从一个端口移动到另一个端口，而无需重新配置 VLAN 。

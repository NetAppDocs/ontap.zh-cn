= 了解 ONTAP System Manager SVM 管理员仪表板
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
从 ONTAP 9.19.1 开始，SVM 管理员可以使用专用的 System Manager 仪表板和 Storage > Volumes 页面来监控和管理他们的 SVM。这在租户 ONTAP 环境中特别有用，其中集群管理员可以为每个租户提供 SVM 仪表板。

群集管理员为 SVM 管理员提供登录凭据。SVM 管理员可以在 SVM 仪表板和*存储 > 卷*页面上查看详细信息和性能。仪表板显示 SVM 协议、网络接口、详细信息和性能指标。卷页面提供有关 SVM 卷的信息，如果群集管理员允许，则提供 SVM 管理员添加新卷的能力。

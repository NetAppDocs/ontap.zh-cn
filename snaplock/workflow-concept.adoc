= SnapLock 工作流
:icons: font
:imagesdir: ../media/


[role="lead"]
在创建 SnapLock 聚合时，您可以指定要使用的 SnapLock 模式：合规性或企业。您创建的 SnapLock 卷将继承此设置。通常，您可以使用文件归档应用程序将文件从主存储移动到 SnapLock 卷。

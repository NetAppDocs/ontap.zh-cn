= 移动 SnapLock 卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
从 ONTAP 9.8 开始，您可以将 SnapLock 卷移动到类型相同的目标聚合，可以是企业到企业，也可以是合规性。要移动 SnapLock 卷，必须为您分配 SnapLock 安全角色。

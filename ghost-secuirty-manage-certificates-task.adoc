= 使用 System Manager 管理证书


[role="lead"]
从 ONTAP 9.10.1 开始，您可以使用 System Manager 管理可信证书颁发机构，客户端 / 服务器证书以及本地（板载）证书颁发机构。阅读有关此任务的更多信息 xref:manage-certificates-sm-task.adoc[此处]。
